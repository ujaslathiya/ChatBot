ChatBot | Lalu Dhabawala

The website is dedicated to "Lalu Da Dhaba," a cloud kitchen entity. Its purpose is to deploy a chatbot named "Lalu Dhabawala" to enhance user experience and streamline interactions. Users have the capability to engage with the chatbot, which primarily offers two key features: placing new orders and tracking existing orders. The chatbot is developed using Dialogflow, utilizing NLP models and ML algorithms. The project utilizes a MySQL database. The backend is constructed using FastAPI, resulting in a comprehensive website that seamlessly integrates front-end, back-end, database, and AI/NLP functionalities.
